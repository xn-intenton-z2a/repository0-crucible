## https://mikemcl.github.io/decimal.js/
Comprehensive API reference and usage guide for Decimal.js, the arbitrary-precision arithmetic library driving all π computations in this project. Essential for correctly setting precision margins, avoiding rounding errors, and optimizing speed in core algorithms. Last updated 2024; highly authoritative as the library’s official site.
