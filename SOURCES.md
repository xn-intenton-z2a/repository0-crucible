# Decimal.js
## https://mikemcl.github.io/decimal.js/
Comprehensive API reference and usage guide for Decimal.js, the arbitrary-precision arithmetic library driving all π computations in this project. Covers precision configuration, rounding modes, performance tuning, and advanced series algorithms (Machin, Gauss–Legendre, Chudnovsky) with extensive usage examples. Essential for precision management, avoiding rounding errors, and optimizing calculation speed.
Last updated 2024; highly authoritative as the library’s official documentation.
