Comprehensive reference to Node.js core modules, covering ECMAScript module resolution, HTTP/HTTPS servers and clients, Streams (including async iterables and chunked transfer), fs/promises for file I/O, URL handling, Performance Hooks, and Worker Threads for parallel computation.
