## https://www.w3.org/TR/skos-reference/

## https://github.com/digitalbazaar/jsonld.js#readme
## https://rdf.js.org/specification/data-model/

## https://github.com/RubenVerborgh/SPARQL.js#readme
## https://github.com/linkeddata/rdflib.js#readme
## https://github.com/rdfjs/N3.js#readme
A unified reference combining the JSON-LD 1.1 Core specification with leading JavaScript libraries and standards for RDF processing in Node.js: `jsonld.js` for JSON-LD operations, the RDF/JS Data Model for abstract RDF interfaces, Comunica federated query engine, SPARQL.js parser, rdflib.js graph toolkit, and N3.js for Turtle/N-Quads parsing and serialization. Essential for constructing, streaming, and manipulating RDF graphs programmatically. Last updated: JSON-LD May 2020; rdf.js Jan 2022; Comunica 2024; SPARQL.js 2023; rdflib.js 2024; N3.js 2024.

# OWL 2 Profiles
## https://www.w3.org/TR/owl2-profiles/

# RDF-star & SPARQL-star Extensions
## https://www.w3.org/TR/rdf-star/
## https://www.w3.org/TR/sparql-star/
Defines the RDF-star model for annotating statements about triples and the SPARQL-star extension for querying RDF-star graphs. Provides grammar for representing reified triples inline, query syntax enhancements for pattern matching on annotated triples, and examples of usage. Useful for advanced ontology features and supporting metadata on relationships in generated JSON-LD OWL artifacts. Last updated: RDF-star Dec 2021; SPARQL-star Oct 2022.
