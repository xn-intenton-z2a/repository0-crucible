# Node.js Core API Reference
## https://nodejs.org/api/
Combined reference for Node.js built-in modules including `fs`, `path`, `url`, `http`/`https`, `process`, `readline`, and core utilities. Covers file I/O operations (loading templates, face definitions), directory creation, environment variables, HTTP server and client functionality, interactive interfaces, signal events (`SIGINT`, `SIGTERM`), standard input/output streams, and utility modules. Essential for implementing CLI features (`--template`, `--output`, `--serve`), HTTP API mode routing, diagnostics (`--diagnostics`), REPL mode, and graceful shutdown. Last updated June 2024; maintained by the Node.js Foundation.
