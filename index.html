repository0-crucible

repository0-crucible is a demo repository that showcases GitHub workflows imported from intentïon agentic‑lib. This repository now includes real example outputs from dry runs of its CLI commands.

To create a self-evolving agentic coding system of your own based on this one, see the TEMPLATE-README.md for more details.

Repository Template

The repository is intended as a template that includes:

  • A Template Base: A starting point for new projects.
  • A Running Experiment: An example implementation that demonstrates one way to use the template.
  • Example GitHub Workflows from agentic‑lib which hand off to reusable workflows.
  • See TEMPLATE-README.md for more details.

Installation

Install via npm:

npm install repository0-crucible

Features

  • CLI tool for managing OWL ontologies.
  • Query, build, refresh, merge, and validate ontology commands.
  • REST API server for ontology operations.
  • Enhanced diagnostic information and verbosity support.

Usage

To run the CLI tool and see help instructions:

node src/lib/main.js --help

When invoked with the --help flag, the CLI displays detailed usage instructions showing available commands and options.

Basic Query:

Command:

Command:
