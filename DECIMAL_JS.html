Usage

Constructor: x = new Decimal(123.4567) y = new Decimal('123456.7e-3')

Precision loss examples: new Decimal(1.0000000000000001) // '1' new Decimal(88259496234518.57) // '88259496234518.56' new Decimal(99999999999999999999) // '100000000000000000000'

Non-decimal string formats: x = new Decimal('2_147_483_647') x = new Decimal('0xff.f') // '255.9375' y = new Decimal('0b10101100') // '172'

Binary conversion: z = x.plus(y) // '427.9375' z.toBinary() // '0b110101011.1111' z.toBinary(13) // '0b1.101010111111p+8'

Exponent notation input: x = new Decimal('0b1.1111111111111111111111111111111111111111111111111111p+1023') // '1.7976931348623157081e+308'
