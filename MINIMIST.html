Digest of MINIMIST

Overview

Minimist is a lightweight Node.js argument parser that converts command-line arguments into an object with keys for flags and values.

Installation

npm install minimist --save

Usage

const minimist = require('minimist')
const argv = minimist(process.argv.slice(2), opts)

API

minimist(args, opts)

  • args (Array): list of command-line tokens (usually process.argv.slice(2)).
  • opts (Object, optional): configuration object with fields: • boolean (Array | boolean) – flags to always treat as booleans or true to treat all non-numeric as booleans. Default: false. • string (Array) – flags to always treat as strings. Default: []. • alias (Object) – mapping of flag names to aliases, e.g. { h: 'help' }. • default (Object) – default values for flags, e.g. { verbose: false }. • unknown (Function): (arg: string) => boolean – called for unrecognized flags; return false to skip setting. • stopEarly (boolean) – stop parsing on first non-option. Default: false. • "--" (boolean) – populate args after "--" into argv['--']. Default: false.

Return Value

ParsedArgs (Object): • _ (Array) – positional arguments. • [flag: string]: boolean | string | number | Array<string | number> • "--" (Array) when opts["--"] is true.

Examples

const argv = minimist(['-x', '3', '--verbose'], {
  boolean: 'verbose',
  default: { x: 0 },
  alias: { v: 'verbose' }
})
// argv = { _: [], x: 3, verbose: true, v: true }
