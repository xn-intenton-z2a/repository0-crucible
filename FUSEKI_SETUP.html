HTTP Endpoints

  • SPARQL Query GET //query Content-Type: application/sparql-query Result: SPARQL XML/Turtle/JSON POST //query Body: SPARQL query Headers: Content-Type: application/sparql-query
  • SPARQL Update POST //update Content-Type: application/sparql-update
  • Graph Store Protocol GET //data?default Accept: application/n-triples POST //data?default Content-Type: text/turtle PUT //data?default Content-Type: text/turtle DELETE //data?default

Best Practices

  • Run with explicit config TTL rather than defaults
  • Use TDB2 for production (tdb2=/path/to/db)
  • Secure endpoints via Apache Shiro: shiro.ini with [users] admin=secret,admin [roles] admin=*

Troubleshooting

  • Inspect logs at FUSEKI_HOME/logs/fuseki.log
  • Enable debug: -Dlog4j.configuration=file:log4j2-debug.xml
  • Check port conflicts: lsof -i:3030
  • Verify Java version: java -version
